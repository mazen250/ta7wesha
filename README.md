# Ta7wesha (تحويشة)

A sleek, privacy-first savings dashboard for tracking your wealth in Egyptian Pounds. Track holdings across 19+ currencies and gold — see everything converted to EGP in real time.

## Features

- **Multi-currency tracking** — Gold (24K/21K/18K), USD, EUR, GBP, SAR, AED, KWD, BHD, QAR, OMR, JOD, TRY, CHF, CAD, AUD, JPY, CNY, INR, and EGP
- **Realtime exchange rates** — Live forex data from AwesomeAPI + gold spot price from gold-api.com, auto-refreshes every 5 minutes
- **Net worth breakdown** — Visual donut chart showing your portfolio composition by currency
- **Goal tracker** — Set savings goals in EGP and track progress with ETA
- **Income planner** — Add multiple income sources in any currency, see surplus and time to goals
- **Expense tracker** — Track monthly expenses with free-text names, see savings rate and surplus
- **Currency converter** — Quick conversions between all 19 supported currencies
- **What-If calculator** — Simulate gold/USD rate changes and extra savings, see projected impact
- **Gold calculator** — Calculate melt value by weight and karat (24K/21K/18K)
- **Zakat calculator** — Automatic nisab check and yearly zakat estimation with disclaimers
- **Export / Import** — Backup and restore all your data as JSON
- **Arabic (Egyptian) support** — Full Egyptian Arabic translation with RTL layout
- **Dark / Light mode** — Toggle between themes
- **Mobile responsive** — Optimized layout for phones, tablets, and desktop
- **Offline-friendly** — Rates and data cached in localStorage, works without internet
- **100% client-side** — No backend, no accounts, no data leaves your browser

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Lucide Icons + React Icons
- AwesomeAPI (realtime forex)
- gold-api.com (gold spot price)
